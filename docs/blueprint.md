# **App Name**: CashCenter.fun

## Core Features:

- Landing Page: Attractive landing page with neon style, offer carousel, and payout logos.
- User Authentication: Secure user registration/login with Google Auth and Email/Password.
- User Dashboard: Personalized dashboard with stats, rewards section, and neon-style navigation.
- Rewards System: Request withdrawals in CASH.
- Admin Payout Management: Admin dashboard for managing payouts, processing payments, and exporting CSV data.
- Offerwall Integration: Secure webhook endpoint for external offerwalls, verifying partner secrets.
- Data Anonymization Tool: Irreversibly anonymizes user data, including email and payout address in all collections and removes user accounts in Firebase Auth as a tool for GDPR compliance.
